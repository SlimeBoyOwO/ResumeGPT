# 基于大模型模型的简历审阅系统 - 个人毕设

## 项目简介

本项目是一个基于大模型模型的简历审阅系统，旨在帮助企业和求职者提高简历审阅的效率和准确性。系统采用大模型模型，通过训练大量简历数据，实现简历的自动审阅、推荐与打分功能。

## 初步设想技术栈

- 前端：Vue3，Typescript，Tailwind CSS，Pinia，Vite
- 后端：Python，FastAPI
- 数据库：MySQL
- 大模型模型：WebLLM
- 辅助模型：BERT模型，Embedding，ChromaDB
- 架构理念：Agent方式的MoE混合专家模型

## 项目功能

- 简历上传与展示
- 简历自动审阅与推荐
